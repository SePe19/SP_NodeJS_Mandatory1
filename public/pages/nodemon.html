Nodemon

What we have learned


 You can install nodemon globally by using npm in the following way:

  npm install -g nodemon


 You can also install nodemon as a development dependency by using npm in the following way:

  npm install --save-dev nodemon


 nodemon wraps your application, and therefore you van run your app by using nodemon instead of node:

  nodemon app.js

 Now your app auto runs, whenever you make changes to your code
