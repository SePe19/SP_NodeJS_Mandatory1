NodeJS

What we have learned


 Use the npm init command to create a package.json file for your application

  npm init

 Now you can edit what you would like to edit, or simply press enter until the file has been created.

 You might want to change index.js to app.js, but it's up to you.


 Now you can create your very first Hello World in NodeJS:

  app.get('/', (req, res) => { res.send('Hello World!')})

 req stands for request and res stands for response.

 Note that req always comes before res.


 Next we need to make the app listen at a certain port:

  app.listen(8080, () => { console.log("The server is running on port:", 8080)})

 We have chosen port 8080 in this example, but you may choose your own.

 you can also create your port outside of app.listen.

 Note that app.listen is always at the very bottom of the JavaScript file.


 When you have created a working app you can run it by writing:

  node app.js

 Or whatever you have called your JavaScript file.
